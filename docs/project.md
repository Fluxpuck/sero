### 2. Application Folder Structure
The project is structured as follows:
```yaml
Main repo
├─ .github/
│   └─ workflows/
│       └─ (CI/CD workflow files)
│
├─ docs/
│   └─ (documentation files)
│
└─ packages/
    ├─ api/
    │   └─ (Restful API files)
    │
    └─ bot/
        └─ (chatbot files)
```
